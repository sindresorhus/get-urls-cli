# get-urls-cli

> Get all URLs in a text file or stdin


## Install

```
$ npm install --global get-urls-cli
```


## Usage

```
$ get-urls --help

  Usage
    $ get-urls <file>
    $ cat <file> | get-urls

  Example
    $ get-urls file.txt
    https://sindresorhus.com
    https://github.com
```
